# my-first-node
A Server Side Rendering Node Js Application

## Usage
```sh
$ run npm install
```
```sh
$ run npm start
# Or Install and run with Nodemon

# Then Visit http://localhost:3000
```
## MongoDB
Create a ".env" file and add your MongoDB URI, local or Atlas
